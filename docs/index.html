Domain Check MCP Server

Official documentation for the Domain Check Model Context Protocol server. Learn how to install, use commands, and explore features.

Available Methods

  • Check Domain Availability
    via IONOS API. Returns availability status and domain pricing information.
  • Get Domain Recommendations
    Fetches related domain suggestions.
  • Fetch Sedo Offers
    Retrieves current offers from Sedo marketplace.
  • Check Domain Pricing
    Supports pricing checks for domains, returning up-to-date pricing details where available.

MCP Server Integration

Add this to your mcp_config.json to use the Domain Check MCP server:

{
  "domain-check-mcp": {
    "command": "npx",
    "args": [
      "-y",
      "domain-check-mcp"
    ]
  }
}

Installation

Install globally via npm or use npx:

npm install -g domain-check-mcp
# or just use
npx domain-check-mcp
