# Installation

Follow these steps to install and run the Domain Check MCP Server:

1. Clone the repo:
```bash
git clone https://github.com/SebastianBoehler/domain-check-mcp.git
```  
2. Install dependencies:
```bash
bun install
```
3. Build the project:
```bash
bun run build
```
4. Start the server:
```bash
bun run start
```
